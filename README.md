# AWS Serverless Cloud Portfolio 🚀

This project demonstrates a fully serverless cloud portfolio website hosted on AWS.

## Live Demo

http://garv-cloud-portfolio-2026.s3-website-us-east-1.amazonaws.com

---

## Technologies Used

* Amazon S3
* AWS Lambda
* API Gateway
* DynamoDB
* HTML/CSS/JavaScript
* IAM

---

## Features

✅ Static portfolio website hosting
✅ Serverless visitor counter
✅ API integration using API Gateway
✅ DynamoDB database integration
✅ Responsive dark-themed UI

---

## Architecture

User → S3 Website → API Gateway → Lambda → DynamoDB
